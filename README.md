# HiTec
App sencilla para inscripción a las diferentes clases.

### Frameworks
- Backend:
    - Supabase
- Framework:
    - React
- Deployment:
    - Vercell

### Ejecutar en local
Clona o copia el repo, después ejecuta
```
npm install
npm run dev
```
